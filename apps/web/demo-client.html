AI 面试陪练官 Demo Client

这个页面现在不仅能测 REST 和 WebSocket，还能直接采集麦克风，把 16k PCM 音频推给 FastAPI，再播放 DashScope 返回的语音分片。

连接配置

如果浏览器里麦克风权限不可用，建议不要直接双击 html， 而是用 VS Code Live Server 或 `python -m http.server` 起一个本地静态服务。
