# Security Policy

## Supported Scope

This repository is currently an actively developed demo / MVP project.

Security issues especially relevant to:

- API keys or credential leaks
- unsafe file upload handling
- websocket authentication or session hijacking risks
- prompt injection paths through uploaded documents or image context

## Reporting a Vulnerability

Please do not open a public issue for sensitive vulnerabilities.

Instead, report privately to the repository owner with:

- affected component
- reproduction steps
- potential impact
- suggested fix if available

The project owner should acknowledge the report, verify the issue, and patch it before public disclosure.
